# Aladdin Copilot（BlackRock）

> 最后更新：2026-04-24
>
> Aladdin 是 **BlackRock 1988 年起内部开发的投资管理平台**——**目前管理全球 $21T 资产**（占全球 AUM 10%+）。**2024-02 Aladdin Copilot 发布**：将 AI / LLM 集成到 Aladdin 平台，让资产经理用自然语言查询风险、做 what-if 分析。**是机构级 AI 金融最典型案例**。

## 一、产品定位

Aladdin Copilot = **"资产管理界的 Bloomberg Terminal + ChatGPT"** —— 对标**机构级金融 AI 应用**。**BlackRock 自己是 Aladdin 最大客户 + 把 Aladdin SaaS 卖给其他资产管理 / 保险公司**。**Copilot 让 Aladdin 从"数据平台" → "对话式决策助手"**。

## 二、核心能力与架构

### 能力
- **自然语言查询**："给我看过去 5 年 SPY 与 我的组合 的相关性"
- **What-if 分析**："如果美联储加 50bps，我的组合 VaR 变化多少"
- **文档理解**：阅读招股书 / 研究报告
- **风险归因**

### 底层
- **与 Microsoft Azure OpenAI 合作**（2024 披露）
- **BlackRock 专有数据微调**
- **40 年 Aladdin 风险引擎** + LLM

## 三、版本与路线图

| 时间 | 里程碑 |
|---|---|
| 1988 | Aladdin 1.0（内部）|
| 1999-2000 | 对外 SaaS 化 |
| 2010s | 成为机构标配 |
| 2023 | **Microsoft + BlackRock AI 合作公告** |
| **2024-02** | **Aladdin Copilot 发布** |
| 2024 年末 | 扩展到 eFront（私募）|
| 2025 | 全球部署 + 更多 LLM 集成 |

## 四、定价与商业化

- Aladdin 订阅：$1-10M/年（视客户规模）
- Copilot：**作为附加功能**
- BlackRock **Aladdin 收入 2024 $1.4B**（占 BLK 总收入 ~7%）

## 五、用户反馈

- **资产经理**：显著提升效率
- **风控团队**：复杂查询快速完成
- 批：**仍谨慎 on 决策 AI**（Human-in-the-loop 必须）
