# 同花顺 · 恒生电子（中国金融 IT 双龙头）

> 最后更新：2026-04-24
>
> 同花顺（300033.SZ）+ 恒生电子（600570.SH）是 **中国金融 IT 双寡头**——前者专攻 **C 端金融资讯 + AI 投研**（"**问财 HithinkGPT**"），后者专攻 **券商 / 基金 / 银行 IT 系统**（O3 / UF / 恒生聚源）。2023 起双双 AI 转型，**借力通义 / DeepSeek** 做金融场景落地。

## 一句话定位

两者 = **"中国金融科技的基础设施"** —— 同花顺 2B + 2C 双打，**问财** AI 投研；恒生电子 B2B IT 系统，**广泛服务中国券商 / 基金 / 银行**。都是**国资参股 + A 股上市** 的传统玩家。

## 三、业务与产品

### 同花顺
- **问财**：自然语言金融查询
- **同花顺 iFinD**：终端（对标 Bloomberg）
- **C 端**：同花顺 App / 电脑版
- **客户**：1 亿 + C 端用户、数万机构

### 恒生电子
- **O3 / UF 券商 IT 系统**（90%+ 中国券商用）
- **LightGPT**：自研金融 LLM（~70B 参数）
- **恒生 AI Lab**
- **客户**：几乎所有中国券商 + 基金 + 部分银行

## 四、技术路线

- 同花顺：接入通义 / DeepSeek + 自研微调
- 恒生电子：**LightGPT 自研**（金融语料 fine-tune）

## 五、商业模式

### 同花顺
- C 端广告 / VIP ¥300-3000/年
- 2B SaaS iFinD
- 2024 营收 ~¥4B

### 恒生电子
- B2B IT 系统 + 订阅
- 2024 营收 ~¥6B

## 六、竞争与壁垒

- 对标 Bloomberg / AlphaSense（国际）、Wind（国内最大）
- 壁垒：**中国合规 + 客户关系 + 监管背景**
- 弱点：AI 技术深度弱于 DeepSeek / 通义自家

## 七、关键风险

- 万得 Wind 竞争（不上市，但 C 端第一）
- DeepSeek / 豆包直接面向终端用户
- 监管风险（金融 AI 备案）

## 八、我的判断

> 1. **同花顺 / 恒生是中国金融 AI "渠道" 寡头** —— 中小券商 / 基金离不开
> 2. **问财 vs Wind 是中国金融信息 C 端之争**
> 3. **恒生电子 LightGPT 成中国金融 LLM 代表**
> 4. **阿里（蚂蚁）对恒生的战略影响** 是关键变量
>
> **我可能错在哪里**：DeepSeek / 豆包零成本 C 端直接吞并同花顺。
