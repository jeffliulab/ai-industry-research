# Replit（公司研究）

> 最后更新：2026-04-24
>
> Replit 是 **2016 年由 Amjad Masad（巴勒斯坦 / 美国）创立的在线 IDE 公司**——**从"浏览器里跑 Python"演化为"AI 全栈 Agent 开发平台"**。2024-10 估值 **$1.25B**（D 轮 $250M）。**50M+ 注册开发者** 是独特资产。**Replit Agent 产品** 见 [产品研究](../../09_AI智能体/12_产品研究/Replit_Agent.md)。

## 一句话定位

Replit = **"50M 开发者社区 + AI Agent 重新出发"** —— 10 年 IDE 积累让 Replit 在 AI 时代有社区 + 部署能力双壁垒。**与 v0 / Bolt / Lovable 同赛道但社区最大**。

## 一、公司速览

- **创立**：2016
- **总部**：San Francisco
- **创始人 / CEO**：Amjad Masad（前 Meta 工程师）
- **员工**：~150
- **估值**：**$1.25B**（2024-10 D 轮 $250M）
- **投资方**：Khosla Ventures、Andreessen Horowitz、ARK、Sequoia、Paul Graham

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2016 | 成立 |
| 2021 | 独角兽 |
| 2023-10 | Ghostwriter AI |
| **2024-09** | **Replit Agent 发布** |
| 2024-10 | D 轮 $250M，估值 $1.25B |
| 2025 | ARR ~$150M |

## 三、业务与产品

- **Replit IDE**：在线编辑 + 运行
- **Replit Deployments**：一键部署
- **Replit Agent**：AI 全栈开发
- **Replit Mobile**：移动端编辑

### 数据
- **50M+ 注册开发者**
- **1 亿 + 项目**
- **全球第二大代码平台**（GitHub 之后）

## 四、技术路线

- Replit Workspace：云 IDE 架构
- **Ghostwriter → Replit Agent**（2024 转型）
- Claude / GPT 混合调用

## 五、商业模式

- **Free + $20 Pro + $33/seat Teams**
- 2025 ARR ~$150M（估算）
- 主要收入：订阅 + 云计算 usage

## 六、竞争与壁垒

- 对标：**Bolt / Lovable / v0** 同赛道
- **GitHub Codespaces**（微软直接对手）
- 壁垒：**50M 社区 + 移动端 + 部署一体**
- 弱点：**代码质量**（相对专业 IDE）

## 七、关键风险

- GitHub Codespaces 压制
- Bolt / Lovable 对 MVP 用户吞并
- 教育市场变化

## 八、我的判断

> 1. **Replit 是 "AI + 全栈" 最平民化的选择**
> 2. **50M 社区是真金白银护城河**
> 3. **2026-2027 IPO 目标 $5B+**
> 4. **Amjad Masad 明星 CEO + 创业者 brand**
>
> **我可能错在哪里**：Bolt + v0 联合生态碾压 Replit。
