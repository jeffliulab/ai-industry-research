# Lovable（公司研究）

> 最后更新：2026-04-24
>
> **注**：产品详见 [AI 智能体板块 Lovable 产品](../../09_AI智能体/12_产品研究/Lovable.md)。本页聚焦公司层面——Lovable 是 **欧洲（瑞典斯德哥尔摩）2023 成立的 AI 全栈开发公司**，**2025-05 ARR $50M，估值 $1.8B**，是**欧洲最快独角兽之一**。

## 一句话定位

Lovable = **"欧洲 AI 互联网代表公司"** —— 在美中双极 AI 格局中证明 **欧洲也能出 $1B+ AI 公司**。**对话驱动开发** 是差异化。

## 一、公司速览

- **创立**：2023-06（原名 GPT Engineer）
- **总部**：Stockholm, Sweden
- **创始人 / CEO**：Anton Osika
- **员工**：~50
- **估值**：**$1.8B**（2025-05 A 轮 $90M）
- **投资方**：Accel、Creandum、Byers Capital、Hummingbird

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2023-06 | **GPT Engineer** 开源（GitHub 热门）|
| 2024-11 | **Lovable 产品 pivot** |
| 2025-05 | **A 轮 $90M，估值 $1.8B** |
| 2025-Q4 | 全球用户破 200 万 |

## 三、业务与产品

- **Lovable 平台**：对话式 AI 开发
- **Lovable Cloud**：托管 + 部署
- **React + Supabase 默认栈**

## 四、技术路线

- Claude 3.5 / 4 Sonnet 主力
- Supabase 深度集成
- shadcn/ui 前端 default

## 五、商业模式

- 订阅 $20-100/月
- 2025 ARR **$50M** → 目标 2026 $120M+

## 六、竞争与壁垒

- 对标：Bolt / v0 / Replit
- 壁垒：**对话式体验 + Supabase 生态 + 欧洲客户**
- 弱点：**创始团队小**、**技术壁垒不硬**

## 七、关键风险

- Bolt / v0 竞争
- Claude API 涨价
- 欧洲融资环境

## 八、我的判断

> 1. **Lovable 是 AI 互联网欧洲代表**
> 2. **非技术创始人 / PM 最佳选择**
> 3. **估值 $1.8B 增速合理**
> 4. **2026-2027 IPO / 收购均可能**
>
> **我可能错在哪里**：Bolt 全栈规模化后 Lovable 掉队。
