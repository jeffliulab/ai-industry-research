# 中美自动驾驶产业对比

> 最后更新：2026-04-30
>
> 自动驾驶 2026 中美格局：**美国 = Waymo + Tesla 两强 + 一批已修正二线**；**中国 = 百度萝卜快跑 + 小马 + 文远 + Momenta 4 巨头 + 一批 ADAS 跨界（华为 / 小鹏 / 蔚来）**。**两国走完全不同路径**——美国靠**科技巨头长期烧钱 + 法规相对宽松**；中国靠**车厂联手 + 城市试点 + 国资托底**。本文做横向产业对比。

## 一句话结论

中美自驾**技术差距 ≤ 12 个月**（Tesla FSD v13 / Waymo / 中国端到端梯队），但**商业模式差距巨大**：美国押注 Robotaxi（Waymo / Cruise 模式）+ 私家 ADAS（Tesla）；中国押注**ADAS（蔚小理 / 华为 ADS）+ Robotaxi 三线推进（萝卜 / 小马 / 文远 / Momenta）**。**中国"车厂主导 ADAS"是独家路径**——美国传统车厂（Ford / GM）已退出，特斯拉一家垄断私家 ADAS。

## 三条关键要点

1. **技术**：中美顶级玩家（Tesla FSD v13 vs 小马 / Momenta 端到端）已基本同代；学术（CVPR / NeurIPS 自驾论文数）中国领先
2. **场景**：美国 Robotaxi 进展更快（Waymo 商业 4 城）；中国 ADAS 进展更快（华为 ADS / 小鹏 XNGP 实测领先）
3. **资本**：美国累计投入 ~$60-80B，中国 ~$30-50B（中国 ROI 部分由车厂业务对冲）

## 一、技术路线对比

| 维度 | 美国 | 中国 |
|---|---|---|
| **L4 Robotaxi 主流路线** | Waymo（高精地图 + 多传感器 + Robotaxi Driver） | 萝卜快跑 / 小马 / 文远（多模态 + 高精地图 + Robotaxi 部署） |
| **L2/L3 主流路线** | Tesla FSD（**纯视觉 + 端到端神经网络**）| 华为 ADS（多传感器）· 小鹏 XNGP（视觉 + 激光雷达）· Momenta（端到端） |
| **激光雷达 vs 纯视觉** | Waymo 多传感器 / Tesla 纯视觉 | 多数中国玩家**多传感器**（含激光雷达），少数纯视觉 |
| **端到端神经网络** | Tesla 2024 全面切换；Waymo 部分采用 | 华为 / 小鹏 / 蔚来 / Momenta 2024-2025 大规模切换 |
| **World Model** | Wayve（英国，被并购意向）· Tesla 内部研究 | 百度 / 小米 / 阿里 等内部研究 |

详见 [自驾技术路线](../01_技术路线/自驾技术路线.md) · [World Model 与 BEV](../01_技术路线/World_Model与BEV.md)。

## 二、商业模式对比

| 维度 | 美国 | 中国 |
|---|---|---|
| **Robotaxi 商业部署城市** | Waymo: Phoenix / SF / LA / Austin / Miami | 萝卜：武汉 / 北京 / 上海 / 深圳；小马：北京 / 广州；文远：广州 |
| **Robotaxi 单价 / 单趟** | $10-30 / 趟 | ¥10-30 / 趟（武汉 ~¥4-16 已可达"低于网约车" |
| **L2/L3 量产车搭载率** | Tesla 全系 + 少数（Mercedes EQS L3） | 华为 / 小鹏 / 蔚来 / 理想 等 NOA 全面铺开 |
| **私家 ADAS 月活订阅** | Tesla FSD ~$99/月 | 华为 ADS ~¥640/月（部分免费）/ 小鹏 XNGP（购车赠送） |
| **货运自驾** | Aurora（已商业化货运）+ Kodiak | 主线挖掘机 / 港口 等 niche 场景 |

## 三、政策与监管

### 美国
- **联邦松散 + 州主导**：加州 / 亚利桑那 / 德州 / 内华达 等先后开放 Robotaxi 试点
- **NHTSA 偏鼓励**：Tesla FSD 投诉调查持续但未停售
- **L3 标准 SAE J3016**：但州级落地分散
- 详见 [全球自驾监管](../03_政策与监管/全球自驾监管.md)

### 中国
- **顶层政策 + 地方试点**：工信部 + 公安部 + 交通部协同；北京 / 武汉 / 深圳 / 上海 等十多个示范区
- **2024-2025 L3 试点**：北京 / 武汉等城市颁发 L3 牌照（蔚来 / 比亚迪 / 长安 等）
- **数据安全 + 跨境**：地图 / 测试数据**强制本土化**——海外玩家进中国受限

## 四、资本与产业

### 美国
- **Waymo + Tesla 两极独占**：累计投入和营收主导
- **二线大幅修正 / 退场**：Cruise / Argo / TuSimple
- **VC 现已大量撤退**

### 中国
- **车厂主导 ADAS**：华为 / 蔚小理 / 比亚迪自研
- **互联网巨头主导 Robotaxi**：百度 / 字节 / 阿里 / 美团（部分）
- **国资 + 沙特资本扩张**：Momenta / 文远 等接受沙特 PIF
- **香港 / 美股 IPO**：小马 / 文远已上市；其他梯队跟进

## 五、用 [BCG Advantage Matrix](../../99_方法论/框架_BCG_Advantage.md) 看

| 国别 + 路径 | 象限 |
|---|---|
| 美国 Robotaxi（Waymo）| **Volume**（赢家通吃；规模效应强）|
| 美国私家 ADAS（Tesla）| **Volume**（车队 + FSD 数据飞轮）|
| 中国 Robotaxi（4 巨头）| **Specialization**（地域 + 城市分化）|
| 中国私家 ADAS（车厂多家）| **Specialization**（按车系 + 价位段）|

## 六、关键差异化的工程深层

1. **数据飞轮**：Tesla 7M+ 车队 vs 中国车厂分散
2. **道路 / 路况**：中国城市道路复杂度（行人 / 电瓶车 / 不规范驾驶）远超美国，是 ADAS 的"加难版"
3. **法规速度**：中国 L3 牌照发放快于美国，但事故责任界定仍模糊
4. **地图 + 数据本土化**：中美**完全不互联互通**，跨国玩家被迫本地化

## 2026 关键变量

1. **Tesla Robotaxi（Cybercab）量产** —— 中美 Robotaxi 直面对决
2. **华为 ADS 4.0 / 小鹏 XNGP 3.0** —— 中国 L3+ 量产车规模化
3. **Waymo 城市数扩展到 10+ ** —— Phoenix 模式可复制性
4. **L4 商业事故 / 监管反弹** —— 任何一起重大事故可能影响整个产业
5. **中美技术 / 数据脱钩进一步加深** —— 影响 Apollo / NVIDIA 等跨国合作

## 我的判断

> 1. **2026-2028 中美自驾会保持"双极化"** —— 不会出现一方碾压另一方
> 2. **Robotaxi 商业化首先在中国 4 城跑通** —— 单价低 + 牌照宽 + 路况数据多
> 3. **L3 私家车量产 中国领先美国 2-3 年** —— 蔚小理 / 华为速度远快于 Mercedes / 福特
> 4. **2030 中美自驾各自占全球 ~40-45%** —— 欧洲 / 日韩 < 15%
>
> **我可能错在哪里**：（a）Tesla Robotaxi 在加州 / 德州大规模放量，美国一家独大；（b）中国出现重大 L3 事故触发严监管，2027-2028 ADAS 增速大幅放缓。
