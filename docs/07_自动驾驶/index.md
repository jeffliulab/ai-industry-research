## 与机器人 / 具身智能的关系

- **技术栈重叠**：端到端神经网络、World Model、传感器融合、数据闭环
- **产业独立**：资本 / 玩家 / 监管 / 商业模式完全不同
- 本站在机器人板块不重复讨论自驾——在该板块下只保留"人形机器人"相关讨论，自驾归此处

!!! note "Tesla 的双重身份"
    Tesla 同时在**机器人（Optimus）** 和**自动驾驶（FSD）** 两个板块被讨论。两边都以"FSD 数据闭环"为切入。
