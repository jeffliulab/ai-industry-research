# 自动驾驶 2026 年度预测

> 最后更新：2026-04-30
>
> 仿 [Benaich State of AI](https://www.stateof.ai/) 体例做"年度可证伪预测 + 自评打分"——自动驾驶板块**12 条预测**，覆盖 Robotaxi 商业化 / L3 量产 / 资本 / 地缘四个维度。

## 12 条 2026 年度预测

### 一、Robotaxi 商业化

**1. Waymo 2026 年内会扩展到 ≥ 7 个商业运营城市**（**70%** 概率）
- 当前：Phoenix / SF / LA / Austin / Miami（5 城）；2026 计划新增 Seattle / DC / Atlanta 等
- 观察：Waymo 官方公告 + Alphabet 财报
- 打分：2027-01

**2. Tesla Robotaxi（Cybercab）2026 年内不会量产交付**（**75%** 概率）
- Musk 承诺 2026 量产，但工程问题众多
- 观察：Tesla 财报 / Cybercab 上路
- 打分：2027-01

**3. 中国 Robotaxi 城市数（萝卜 + 小马 + 文远 等合计运营城市）≥ 15 个**（**75%** 概率）
- 当前 ~10 个城市
- 观察：工信部 / 中国汽车工业协会数据
- 打分：2027-01

### 二、L3+ 私家车

**4. 中国 ≥ 3 家车厂在 2026 出货 L3 量产车（合计 ≥ 50,000 台）**（**60%** 概率）
- 蔚来 / 长安 / 比亚迪等已申请 L3 牌照
- 观察：中汽中心销量数据
- 打分：2027-Q1

**5. Mercedes / BMW / 福特等美欧传统车厂 L3 量产车销量 < 100,000 台合计**（**80%** 概率）
- 美欧 L3 推进慢于中国
- 观察：各家财报销量披露
- 打分：2027-01

**6. Tesla FSD v14 / v15 在 2026 内推出**（**85%** 概率）
- Tesla 历来频繁迭代
- 观察：Tesla 官方更新
- 打分：2026-Q4

### 三、资本与公司

**7. Aurora 2026 内会有 ≥ 3 家货运客户商业部署货运自驾**（**60%** 概率）
- 与 Volvo 等已合作
- 观察：Aurora 季报
- 打分：2027-01

**8. 至少一家中国 Robotaxi 公司（萝卜 / 小马 / 文远 / Momenta）2026 EBITDA 转正**（**40%** 概率）
- Robotaxi 商业化早，盈利仍困难
- 观察：上市公司财报
- 打分：2027-Q2

**9. Tesla FSD 月活订阅用户 2026 末 ≥ 500 万**（**55%** 概率）
- 当前 100-200 万级（推算）
- 观察：Tesla 财报披露
- 打分：2027-01

### 四、地缘 / 监管

**10. 美国不会在联邦层面统一 L3/L4 法规** —— 仍是州主导（**90%** 概率）
- 观察：NHTSA 联邦立法进度
- 打分：2027-01

**11. 中国会在 2026 年内向 ≥ 3 家车厂正式发放 L3 商业化牌照（非试点）**（**65%** 概率）
- 观察：工信部 / 公安部公告
- 打分：2027-01

**12. 至少一起 Robotaxi 重大事故（致死或严重）会触发某个城市暂停运营**（**50%** 概率）
- 历史事故率：Cruise SF 2023 已先例
- 观察：NHTSA / 中国工信部 / 媒体报道
- 打分：2027-01

## 重大不确定性

- Musk / Tesla 战略变更（如把 Optimus 优先于 Cybercab）
- 中美技术 / 数据脱钩升级（地图 / 高精传感数据出口管制）
- 大型自驾事故触发监管反弹
- 经济衰退导致 Robotaxi 投资减速
