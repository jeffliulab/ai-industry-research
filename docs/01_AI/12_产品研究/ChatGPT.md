# ChatGPT（含 Codex / Operator / Agent）

> 最后更新：2026-04-24
>
> ChatGPT 是**人工智能历史上最快达到 1 亿用户的消费产品**——2022-11 发布后 2 个月破 1 亿 MAU。2025 周活 ~8 亿，Plus / Pro / Team / Enterprise 多层订阅 + Agent（Operator）+ Canvas + GPTs 生态形成"AI 超级入口"。OpenAI 2025 ARR $4B+，ChatGPT 占 >70%。

## 一、产品定位

ChatGPT 是 **"AI 时代的 Google + Slack + Excel 入口"**——不再只是对话机器人，而是**工作流中枢、搜索替代、Agent 容器、API 分发点**。对 OpenAI 而言，ChatGPT 是从 API 公司转向 C 端 + 企业"全栈产品公司"的核心资产。

## 二、核心能力与架构

### 核心能力
- **对话**：基于 GPT-4o / GPT-5 / o3 等模型
- **推理模式（o-series）**：深度思考
- **视觉 / 图像生成**：DALL-E 3 / GPT-4o 图像
- **语音 Advanced Voice**
- **Canvas**：边对话边编辑文档 / 代码
- **GPTs**：用户自建定制版本
- **Operator / Agent**：操作浏览器 / 桌面
- **Deep Research**：多步检索生成报告
- **Projects / Memory**：多轮会话状态

### 底层模型
- **GPT-4o**：多模态默认
- **GPT-4.1 / GPT-4.5 / GPT-5**：旗舰
- **o1 / o3 / o3-mini**：推理模型
- **GPT-4o mini**：低成本

## 五、用户反馈

### 消费者反馈
- **"Google 的真正挑战者"**：2024-2025 搜索行为数据迁移明显
- **Memory / Projects 功能好评**
- **Advanced Voice 被称为 "Her 最接近版"**
- **Operator 目前不稳定**（2025-Q1 beta 反馈）

### 企业反馈
- **Enterprise 采购飞涨**：2024-2025 Fortune 500 采用率 80%+
- **SOC2 / HIPAA 等合规已满足**
- **对比 Claude**：ChatGPT 品牌强 + 多模态 + 用户熟悉度
- **"过度对齐"** 不如早期："Sorry I can't help with that" 频率变高

### 批评
- **事实性问题**：仍有 hallucination
- **"feel dumber after GPT-5"**：部分用户认为"GPT-5 比 GPT-4o 不如"
- **依赖 OpenAI 基础设施**：宕机时影响巨大

## 七、使用笔记

### ChatGPT 最适合
1. **通用 C 端**：问答 / 写作 / 学习
2. **多模态任务**：图像理解 + 生成 + 语音
3. **GPT Store 应用**：定制 Agent
4. **Deep Research**：深度研究报告

### 不太适合
- **严肃编程**：Cursor / Claude Code 更好
- **长文档（>200K）**：Gemini 2M 窗口更强
- **本地化 / 数据不出境**：需 Enterprise 版或转到 Azure
