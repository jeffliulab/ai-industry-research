# 中国 AI 梯队结构

> 最后更新：2026-04-23
>
> 2023 年 "中国大模型六小虎" 是高光叙事；2025 年分化严重；2026 年的格局是 **DeepSeek 一马当先 + 三大厂 + 梯队二 + 一批掉队**。本文按**真实产业地位**（不是融资数字）划分中国 AI 公司的梯队。

## 一句话结论

中国 AI 已从"六小虎齐飞"分化为**差异化分工的产业图景**：DeepSeek 主攻技术前沿 + 开源冲击波；字节 / 阿里主攻 C 端应用；Moonshot / 智谱走订阅订阅 + 企业；其他几家正在被迫转型或被吸收。

## 三条关键要点

1. **DeepSeek 独立成为"第 0 梯队"**：R1 开源 + 低成本冲击击穿国际定价；2025 下半年起技术影响力已超越其他中国公司
2. **大厂 AI 重回主导**：字节豆包 / 阿里通义 / 腾讯混元在 C 端应用 + 企业云 + 生态上对初创公司形成降维打击
3. **六小虎分化加剧**：Moonshot 转向 C 端订阅做对了；智谱死磕 B 端；MiniMax 押 Talkie 等消费；百川 / 阶跃 / 零一万物在缩小规模

## 梯队划分

### 第 0 梯队 · 技术冲击者
- **DeepSeek**（幻方量化 / 高瓴）
  - V3 / R1 开源，技术含量全球顶级
  - 2025 年训练成本 + 模型效率工程领先
  - 不追求 C 端产品，纯研究 + 开源品牌
  - 详见 [DeepSeek 公司调研](../11_公司研究/DeepSeek.md)

### 第 1 梯队 · 三大厂
- **字节 · 豆包 / Seed**
  - 豆包 C 端 MAU 已超 7000 万（2025 末），国内头部
  - Seed 模型研究团队技术过硬
  - 火山引擎企业 AI 平台
- **阿里 · 通义 / Qwen**
  - Qwen 系列全球最受欢迎的中文开源模型之一
  - 阿里云 AI 收入 2025 加速
  - 通义千问消费端不温不火
- **腾讯 · 混元**
  - 混元大模型 + 混元图像 + 混元视频
  - 主要绑定腾讯生态（微信 / QQ / 游戏）
  - 公开产品化节奏比字节慢

### 第 2 梯队 · 头部初创（分化中）
- **Moonshot · 月之暗面**
  - Kimi C 端订阅转型成功（2025 下半年）
  - Kimi 探索版深度研究模式好评
  - 2026 Q1 K2 发布
  - 详见 [Moonshot 公司调研](../11_公司研究/Moonshot.md)
- **智谱 AI · GLM**
  - B 端企业市场领先（金融 / 政企客户）
  - GLM-4.5 + CodeGeeX 代码模型
  - 开源路线积极
- **MiniMax**
  - Talkie 等 C 端社交产品全球化
  - abab 模型用于社交场景
  - 技术与商业两条腿走

### 第 3 梯队 · 挑战加剧
- **百川智能**（王小川）
  - Baichuan 系列稳定但缺乏突破
  - 转向医疗 + 金融垂直
- **阶跃星辰 · Step**
  - 多模态 Step 模型
  - 技术稳健但商业化节奏较慢
- **零一万物**（李开复）
  - 2025 年资源收缩，转向 2B

### 第 4 梯队 · 垂直 / 新势力
- **DeepRobotics · 宇树 · 智元** —— 机器人 / 具身，见 [机器人板块](../../02_机器人/11_公司研究/)
- **银河通用 · 千寻智能 · 穹彻智能** —— 具身智能，见 [具身智能板块](../../03_具身智能/11_公司研究/)
- **Manus / Monica** —— Agent 产品
- **Kimi Coder / Seed Code** —— 各家的 coding 产品

## 用 Porter 五力看中国 AI 行业

### 现有竞争者
极度拥挤：六小虎 + 三大厂 + DeepSeek + 各垂直创业公司 = 30+ 玩家

### 新进入者威胁
降低：**训练成本门槛**从 2023 的 $10M 提升到 2025 的 $100M+（除非像 DeepSeek 那样算法创新）

### 供应商议价力
**极强**：
- NVIDIA 高端卡被美国出口管制限制 → 昇腾 / 阉割版 H20 等次优选择
- 电力 / 数据中心容量国内相对充裕

### 买方议价力
**升高**：
- 企业客户越来越懂行，不再盲目采购
- 价格透明化（豆包 / DeepSeek 等低价倒逼）

### 替代品威胁
**中**：
- 开源模型越来越强，自部署成替代
- 海外 API 通过 VPN 或合规渠道也是替代

## 用 VRIO 看中国 AI 公司的真资源

（参考 [VRIO 框架](../../99_方法论/框架_VRIO.md)）

| 公司 | 真正 VRIO 资源 |
|---|---|
| **DeepSeek** | 算法工程 efficiency · 开源品牌 · 幻方背景 |
| **字节** | 7B+ DAU（抖音 / 今日头条）· 全球化基础设施 · 算法 |
| **阿里** | 阿里云渠道 · 电商大数据 · Qwen 开源生态 |
| **腾讯** | 微信生态（10B+ MAU）· 游戏场景 |
| **Moonshot** | C 端品牌（Kimi）· 长上下文技术 |

**中国 AI 的独特 VRIO**：
- 政策访问 + 国家算力 + 数据环境（国际公司不可复制）
- 工程师供给与成本效率

## 2026 关键变量

1. **DeepSeek V4 / R2** 能否保持对国际前沿的冲击
2. **字节 vs OpenAI 在 C 端的全球 / 国内对峙**：豆包能否反攻 ChatGPT 在中文世界的缺席
3. **六小虎中再有一家退出基础模型**：2026 上半年可能发生
4. **中美算力差距**：Blackwell / Rubin 出口管制 vs 昇腾产能爬升
5. **本土大模型备案 + 安全审查**：新一轮更严监管可能到来
