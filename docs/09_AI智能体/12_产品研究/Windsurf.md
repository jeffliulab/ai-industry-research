# Windsurf

> 最后更新：2026-04-24
>
> Windsurf 是 **Codeium（2021 成立，原做 Copilot 免费替代）** 于 2024-11 发布的 **AI 原生 IDE**——和 Cursor 一样 fork 自 VS Code，**Cascade Agent** 是其杀手功能。2025-05 **Google $2.4B 收购 Codeium 创始团队**（类似 Character.AI 模式），Windsurf 产品继续独立运营。

## 一、产品定位

Windsurf 是 **"Cursor 的直接对手 + Agent-first IDE"**——与 Cursor 同样 fork VS Code，但更强调**自主 Agent（Cascade）** 和 **企业 / Enterprise 能力**。主打"**Flow state editor**" —— 从补全到 Agent 的无缝流动。

## 二、核心能力与架构

### 核心能力
- **Cascade Agent**：多步自主编程 Agent（最大差异化）
- **Supercomplete**：多行智能补全
- **Command**：类似 Cursor Cmd-K
- **Inline AI / Chat**：侧边栏对话
- **Memory**：项目记忆 + 用户偏好

### Cascade 特色
- **自动读文件 + 编辑 + 运行**
- **Cascade Flow**：长任务自动化
- **"Write mode" + "Chat mode"** 分离

### 模型层
- 调用 Claude 3.5 / 4
- GPT-4o / GPT-5
- Gemini 2.5
- 自家 Cascade 模型（fine-tune）

## 三、版本与路线图

| 时间 | 里程碑 |
|---|---|
| 2021 | Codeium 成立 |
| 2022-2024 | Codeium 做 GitHub Copilot 免费替代（浏览器插件）|
| 2024-11 | **Windsurf 发布**（VS Code fork）|
| 2024-12 | Cascade Agent 扩展能力 |
| 2025-01 | Windsurf 1.0 正式版 |
| 2025-03 | Windsurf Enterprise |
| **2025-05** | **Google 以 $2.4B 收购创始人 + 顶级团队**（类似 Character.AI 技术授权）|
| 2025-07 | Cognition AI 收购 Windsurf 产品团队（剩余部分）|
| 2025-Q4 | Windsurf 独立运营 |

### 2025-05 奇怪事件
- **Google 收购 Codeium CEO Varun Mohan + 核心研究员**（$2.4B 技术授权）
- **OpenAI 被传初期出价 $3B 收购但失败**
- **剩余 Windsurf 产品 + 员工**被 Cognition AI（Devin 公司）收购继续运营
- 这是 2025 最奇异的 AI 创业公司命运

## 四、定价与商业化

### 订阅
| 层级 | 月费 | 配额 |
|---|---|---|
| **Free** | $0 | 基础功能 |
| **Pro** | $15 | 500 Flow Action Credits |
| **Ultimate** | $60 | 更多配额 + 更快模型 |
| **Enterprise** | 自定义 | SOC2 + 私有 |

**比 Cursor 便宜 5 美元** —— 曾是关键差异化

### 商业化挑战
- Google 收购创始人后，产品未来不确定
- 2025-Q4 Cascade 模型训练团队已大部分流失
- Enterprise 销售受拖累

## 五、用户反馈

### 正面
- **Cascade Agent 体验**曾超越 2024 年 Cursor
- **Supercomplete** 补全质量好
- **免费版足够日常开发**

### 批评
- **Google 收购事件**打击社区信心
- **创始人离开后更新变慢**
- **和 Cursor 同质化**，差异化缩小

### 历史评价
- 2024-Q4 到 2025-Q1 与 Cursor 并列的 AI IDE 双雄
- 2025-05 事件后掉队

## 七、使用笔记

### 最适合
1. **Cursor 用户想尝试替代**
2. **Cascade Agent 工作流**（类似 Composer）
3. **成本敏感用户**（比 Cursor 便宜）
4. **Codeium 老用户**的自然升级

### 风险
- **创始团队流失** 后产品路线不确定
- **Cognition AI 收购后战略可能大变**
- **DevRel 和社区支持减弱**
