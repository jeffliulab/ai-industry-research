# AI 智能体（Agents）

> 最后更新：2026-04-24
>
> **"AI 智能体"板块聚焦 Agent 产品形态** —— Coding Agent · Computer Use · Multi-Agent · 企业 Agent。不重复基础模型本身（Claude 家族 / GPT / Gemini 在 [人工智能](../01_AI/index.md) 板块），而是聚焦"让模型去做事"的产品层级。

## 一句话定位

**Agent = 让大模型从回答问题变成执行任务的产品形态**——2023 AutoGPT 点燃叙事 → 2024 Claude Computer Use / Devin 证明可行 → 2025 Claude Code / ChatGPT Agent / Cursor / OpenAI Codex CLI 爆发 → 2026 "Agent 生产化"进入企业 B 端主战场。

## 2026 关键判断（提要）

1. **Coding Agent 是当前最成熟赛道**：Cursor ARR $500M、Claude Code 增长爆炸、Devin 进入真实工作流
2. **Computer Use 仍在早期**：Claude CU 2024-10 首发，Operator 2025-01 beta，生产化 2026-2027
3. **企业 Agent 市场 2026 起爆**：Sierra / Harvey / Hebbia 按"outcome pricing"重写 SaaS 定价
4. **"单一 JTBD Wrapper"必死**：基础模型 Agent 能力飞速提升，薄层 Agent wrapper 无法持久
5. **MCP 协议成为事实标准**：Anthropic 2024-11 开源的 Model Context Protocol 正被全行业采用
