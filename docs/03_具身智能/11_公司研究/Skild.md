# Skild AI

> 最后更新：2026-04-24
>
> Skild AI 是 **CMU 2023 年 spinoff** 的具身智能基础模型公司——由 **Deepak Pathak**（CMU 教授）+ Abhinav Gupta 创立，专注 **通用机器人大脑（Robot Foundation Model）**。2024-07 **$300M A 轮融资，估值 $1.5B**（Jeff Bezos、SoftBank 参投），**是 Physical Intelligence 之外最受瞩目的 RFM 公司**。

## 一句话定位

Skild = **"通用具身智能大脑 + 不做硬件"** —— 类似 Physical Intelligence 的"做模型不做身体"路线，目标让任何机器人都能用 Skild 大脑。**深度学习 + 真机 + 仿真 + 互联网视频**多源数据策略。

## 一、公司速览

- **创立**：2023
- **总部**：Pittsburgh, PA（CMU）
- **创始人**：**Deepak Pathak**（CEO）、Abhinav Gupta
- **员工**：~50
- **估值**：**$1.5B**（2024-07 A 轮 $300M）
- **投资方**：Lightspeed、Coatue、Bezos Expeditions、SoftBank、SIP、Amazon、Sequoia

## 三、业务与产品

- **Skild Brain**：通用机器人基础模型
- **目标合作**：机器人硬件公司嵌入 Skild Brain
- **不自研硬件**

## 四、技术路线

- **Data-centric**：真机 + 仿真 + 互联网视频
- **End-to-end VLA**
- Deepak Pathak 学术派 + 工程导向
- 主张"一个模型操作千种机器人"

## 五、商业模式

- 企业 B2B 授权
- 早期无公开定价
- **类似 OpenAI API 模式** 但面向机器人

## 六、竞争与壁垒

- 对标：**Physical Intelligence (π)** 最直接
- 壁垒：**CMU 学术 + Bezos 支持 + 顶级数据团队**
- 弱点：需要硬件合作伙伴

## 七、关键风险

- π 技术 / 商业化领先
- NVIDIA GR00T / Google Gemini Robotics 大厂挤压
- 机器人公司自研大脑趋势
