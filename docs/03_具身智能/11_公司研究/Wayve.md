# Wayve

> 最后更新：2026-04-24
>
> Wayve 是 **英国伦敦 2017 年 Cambridge PhD 创立的自驾 + 具身 AI 公司**——**Alex Kendall 主创**，**端到端学习（不用 HD Map）** 路线的倡导者。2024-05 **C 轮 $1.05B**（SoftBank / NVIDIA / Microsoft 领投），**估值 $2.5B**，成为**欧洲最大 AI 独角兽之一**。2025 与日产 / Uber 合作。

## 一句话定位

Wayve = **"自驾界的 DeepMind + 端到端主义"** —— 10 年坚持 **纯视觉 + 端到端神经网络**（不用激光雷达 + HD 地图）路线，**2024 巨资完成验证**。被 **Alex Kendall 称为 "Embodied AI 第一家"**。

## 一、公司速览

- **创立**：2017
- **总部**：伦敦（双总部还含 Mountain View、温哥华、东京）
- **创始人**：**Alex Kendall**（CEO）、Amar Shah
- **员工**：~250
- **估值**：**$2.5B**（2024-05 C 轮 $1.05B）
- **投资方**：SoftBank Vision Fund 2、NVIDIA、Microsoft、Eclipse、Virgin Hyperloop

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2017 | 剑桥 PhD spinoff |
| 2019 | 首次公开道路测试 |
| 2022 | B 轮 $200M |
| 2023 | GAIA-1 世界模型发布 |
| 2024-03 | LINGO-2 视觉语言驾驶模型 |
| **2024-05** | **C 轮 $1.05B，估值 $2.5B** |
| 2024-Q4 | 日产 / Uber 合作 |
| 2025-Q3 | 量产车 OEM 合作推进 |

## 三、业务与产品

- **Wayve AI Driver**：端到端自驾大脑
- **GAIA-1 / LINGO-2**：世界模型 + 视觉语言
- **OEM 合作**：日产、Uber、多家欧洲车企

## 四、技术路线

- **纯视觉 + 端到端**
- **没有 HD 地图**
- **世界模型** GAIA 是关键研究成果
- 数据 + 仿真双轮

## 五、商业模式

- OEM 授权（类似 Mobileye）
- 数据 + 模型出售
- 未公开 ARR

## 六、竞争与壁垒

- **自驾对标**：Tesla FSD（同端到端派）、Mobileye（传统派）、Waymo（L4）
- **具身 AI**：Skild、Physical Intelligence
- 壁垒：**欧洲主权 AI + 10 年端到端坚持**
- 弱点：**没有自建车队**，依赖 OEM

## 七、关键风险

- Tesla FSD 端到端规模化碾压
- Mobileye 老牌威胁
- OEM 合作落地慢

## 八、我的判断

> 1. **Wayve 是"端到端自驾" 的欧洲代表** —— 与 Tesla 同路线但规模不敌
> 2. **具身 AI 扩展是关键**：不只自驾，还做通用具身模型
> 3. **被 NVIDIA / Microsoft / SoftBank 投入意味深长**
> 4. **终局**：IPO 或被 OEM / 大厂整合
>
> **我可能错在哪里**：日产合作成功后 Wayve 成为欧洲自驾事实标准。
