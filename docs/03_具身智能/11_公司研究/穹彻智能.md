# 穹彻智能

> 最后更新：2026-04-24
>
> 穹彻智能（Noetix Robotics）是 **2023 成立的中国具身智能公司**——由 **卢策吾（上海交大教授）** 创立。专攻 **触觉 + 视觉多模态具身**，2024 A+ 轮 ¥500M+，估值 ~$300M。与 **蚂蚁 / 复星** 深度绑定，主打**柔性操作 + 触觉大模型**。

## 一句话定位

穹彻 = **"触觉优先的具身 AI"** —— 卢策吾团队 10 年上交大触觉感知积累，**加入触觉作为"第四模态"** 是其差异化。偏**精细操作场景**（医疗、实验室、零售）。

## 一、公司速览

- **创立**：2023
- **总部**：上海
- **创始人 / CEO**：卢策吾（上交大）
- **员工**：~100
- **估值**：~$300M（2024 A+ 轮）
- **投资方**：蚂蚁、复星、启明、红杉中国、真格

## 三、业务与产品

- **AnyTouch**：触觉 + 视觉 + 语言基础模型
- **应用场景**：实验室操作、医疗、零售
- **合作方**：蚂蚁（支付宝场景）

## 四、技术路线

- **触觉大模型**（差异化）
- **视觉 / 语言 / 触觉**三模态
- 上交大机器学习组积累

## 五、商业模式

- B2B 模型 + 解决方案
- 早期 ARR ¥30-50M
- 长期：触觉模型授权

## 六、竞争与壁垒

- 对标：银河通用、千寻智能
- 壁垒：**触觉 + 上交大学术 + 蚂蚁 / 复星 产业资源**
- 弱点：规模小

## 七、关键风险

- π / Skild 国际领先
