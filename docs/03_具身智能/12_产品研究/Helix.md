# Helix（Figure）

> 最后更新：2026-04-24
>
> Helix 是 **Figure 2025-02 发布的自研 VLA 模型**——**与 OpenAI 2024-02 合作后，Figure 2025 脱离 OpenAI 独立自研**。**双系统架构**：**System 1（快速 80Hz 控制）+ System 2（慢速 7-9Hz 推理）**。是 Figure 02/03 的大脑。

## 一、产品定位

Helix = **"Figure 独立自主的 VLA 大脑"** —— **双速系统分离推理与控制**，让 Figure 02 能 **"听自然语言 + 实时执行"** 工业任务。**视频 demo 惊艳** 但生产效果待验证。

## 二、核心能力与架构

### 双系统
- **System 1（80Hz）**：快速动作控制（Flow Matching）
- **System 2（7-9Hz）**：高层规划 + VLM 理解
- **通信**：System 2 输出 latent vector 给 System 1

### 能力
- **视觉 + 语言 + 动作** 端到端
- **实时工厂环境**：BMW 试点
- **跨任务泛化**：一个模型，多种任务
