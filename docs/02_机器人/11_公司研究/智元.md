# 智元机器人（AgiBot）

> 最后更新：2026-04-24
>
> 智元是**中国人形机器人最热门公司**——2023-02 由"稚晖君"彭志辉（华为前天才少年、B 站百万粉丝）创立，一出生便自带流量。2024-08 A 轮估值 $1.5B+，2025 估值传闻 $3B+。产品 **远征 A1 / A2 / 灵犀 X1**，**量产进度中国第一**。

## 一句话定位

智元 = **"中国 Figure"** —— 背靠稚晖君流量 + 华为资源 + 高瓴 / 鼎晖等重金，定位**工业 + 商业服务人形**。2024-2025 **率先量产 1000+ 台** 是中国人形领跑者。

## 一、公司速览

- **创立**：2023-02
- **总部**：上海 + 深圳
- **联合创始人**：**彭志辉（稚晖君）**、邓泰华、彭志刚
- **员工**：~500
- **估值**：**$3B+**（2025 传闻）
- **投资方**：高瓴、红杉中国、鼎晖、腾讯、比亚迪、上汽、宁德时代

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2023-02 | 成立 |
| 2023-08 | **远征 A1 发布**（首款）|
| 2024-08 | A 轮 $1B+，估值 $1.5B |
| 2024 | 远征 A2、灵犀 X1 发布 |
| 2025-03 | 全球首个 1000 台量产纪录 |
| 2025-Q3 | 传 $3B 估值新一轮 |

## 三、业务与产品

- **远征 A1 / A2**：工业人形
- **灵犀 X1**：消费 / 商业服务（开源部分）
- **G1 数据采集**：数据飞轮产品
- **客户**：上汽、比亚迪、宁德时代工厂

## 四、技术路线

- **电动 + 模块化**
- **自研大小脑双系统**（VLA 方向）
- **数据飞轮**：遥操作 + 仿真 + 真机
- **开源 AgiBot World 数据集**（2024-12）

## 五、商业模式

- 2024-2025 以 B 端销售 + 试点为主
- 定价 ¥10-30 万
- 2025 收入估算 ¥100-300M

## 六、竞争与壁垒

- 对标：Tesla Optimus、Figure、宇树、优必选
- 壁垒：**稚晖君品牌 + 量产能力领先**
- 弱点：**AI 大脑仍追赶 Figure / Apptronik**

## 七、关键风险

- 美国制裁（虽未被列入，有潜在风险）
- 宇树等对手四足 + 人形双线竞争
- 稚晖君个人依赖度高

## 八、我的判断

> 1. **智元是 2025-2026 中国人形最大赢家候选**
> 2. **量产能力领先**让其在中国订单上占优
> 3. **上汽 / 比亚迪 / 宁德时代三个超级客户** 是结构性护城河
> 4. **估值 $3B 合理**：对应 2026 目标量产 5000 台
> 5. **终局：IPO 或被产业资本整合**
>
> **我可能错在哪里**：稚晖君如退出则公司光环骤减；特斯拉 Optimus 量产冲击过早。
