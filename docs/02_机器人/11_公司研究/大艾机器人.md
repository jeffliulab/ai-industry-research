# 大艾机器人

> 最后更新：2026-04-29
>
> 大艾是 **北京 2014 创立的医疗康复外骨骼公司**——北航（北京航空航天大学）系。**帅梅**（北航机械工程教授）创立。**艾动 / AiLegs 系列** 是中国最早通过 NMPA 二类医疗器械注册的下肢康复外骨骼之一（2017）。**主战场**：脊髓损伤 + 中风康复，**军民融合背景** —— 部分产品涉及军用康复 / 退伍军人。**与程天 + 傅利叶 ExoMotus 构成中国康复外骨骼三巨头**。

## 一句话定位

大艾 = **"中国康复外骨骼的工程派"** —— 北航学术血统 + **军民融合 + 残联医保渠道**。**艾动**（医院款）+ **艾家**（家用款）双线，覆盖康复 + 个人辅具。**比程天更早 NMPA 批准 + 比傅利叶更专注医疗** —— 是国产外骨骼"老大哥"位置但商业化推进相对慢。

## 三、业务与产品

- **AiLegs 艾动**（核心）：医院 / 康复诊所下肢外骨骼
  - 脊髓损伤完全 / 不完全 患者康复
  - 中风偏瘫患者步态训练
- **AiHome / 艾家**：家用个人款
- **儿童康复款**：脑瘫 / 发育异常儿童
- **退伍军人康复**：军民融合渠道（军方 / 武警康复中心采购）

## 四、技术路线

- **力 + 关节角度传感** + 步态控制
- **轻量化金属结构**（北航航空材料背景）
- 控制路径**与 Ekso EksoNR 类似**（不是 Cyberdyne 的 EMG 路线，也不是 Wandercraft 的全自主平衡）

## 五、商业模式

- **B2B 公立医院 + 残联 / 退伍军人康复机构**（核心）
- **B2C 家用艾家** （少量）
- **政府 / 军方采购**渠道
- **价格区间**：与程天类似，¥30-60 万 / 套（机构）

## 七、竞争与壁垒

- **直接对手**：程天 · 傅利叶 ExoMotus · 国际玩家
- **壁垒**：
  - **NMPA 最早批准之一**（2017）——首发优势 + 临床数据积累
  - **军民融合渠道**（武警 / 退伍军人康复中心）—— 是其他玩家进不去的
  - **北航学术 + 帅梅个人 IP**
- **弱点**：
  - **品牌声量**低于傅利叶（傅利叶有人形 + 沙特国际新闻加持）
  - **商业化推进**相对慢
  - 家用艾家路径**比 ReWalk Personal 还难**——中国家庭付费意愿 + 报销支持有限

## 八、关键风险

- **中国康复医疗政策不变**：医保覆盖不足 + 退伍军人渠道有上限
- **程天 / 傅利叶 拉开差距**：竞争对手融资 / 国际化更快
- **创始人接班**：帅梅 学者型创始人，企业管理迭代风险

## 九、我的判断

> 1. **大艾是中国康复外骨骼"老大哥"** —— 历史功绩（最早 NMPA）+ 军民融合差异化，但**商业化执行偏弱**
> 2. **退伍军人康复是独家渠道**——程天 / 傅利叶进不去；这是大艾长期价值锚
> 3. **2026-2028 是大艾追赶 / 被超越的关键期** —— 看公司能否在融资 + 国际化上跟上程天 / 傅利叶节奏
> 4. **可能被国资 / 央企并购** —— 军民融合 + 医疗康复刚需，是央企整合的合理对象
>
> **我可能错在哪里**：（a）国家政策对国产医疗器械倾斜，让大艾享受替代红利；（b）退伍军人康复市场被新进入者打开，大艾失去渠道独家性。
