# Ekso Bionics

> 最后更新：2026-04-29
>
> Ekso Bionics 是 **2005 加州 Berkeley 出身的医疗外骨骼公司**——前身 Berkeley Bionics，**最早从 DARPA 军用项目（HULC）转做民用康复**。**EksoNR**（中风 / SCI 康复）+ **EksoUE**（上肢康复）是核心产品。**主战场是医院 / 康复诊所 B2B**，不像 ReWalk 推家用 B2C。**2014 NASDAQ 上市（EKSO）**，市值历经低谷。

## 一句话定位

Ekso = **"医院诊所外骨骼" 代表** —— 不卖给个体患者，专做**临床康复机构市场**。**EksoNR FDA 批准用于中风急性期康复**（与 ReWalk Personal 6.0 路径明显分化）—— Ekso 选了"康复设备"，ReWalk 选了"家用辅具"，前者更稳，后者更难。

## 一、公司速览

- **创立**：2005（前身 Berkeley Bionics）
- **总部**：San Rafael, California
- **创始人**：Russ Angold + Homayoon Kazerooni（UC Berkeley）
- **现 CEO**：Scott Davis（2023-）
- **员工**：~80-100
- **上市**：NASDAQ: EKSO
- **市值**：~$30-60M 区间（2025）
- **官网**：eksobionics.com

## 三、业务与产品

- **EksoNR**（核心）：中风 / SCI 急性期康复（医院使用）
- **EksoUE**：上肢康复（中风后上肢功能恢复）
- **EksoVest**（已剥离 / 重组）：工业上肢辅助（曾与福特合作）

## 四、技术路线

- **力 + 惯性传感**控制（与 Cyberdyne EMG 路线不同）
- **可调节身体尺寸 + 多模训练模式**（被动 / 辅助 / 主动）
- **数据采集 + 治疗师可视化**：医院端 SaaS 软件辅助康复方案设计

## 五、商业模式

- **B2B 医疗机构销售 + 租赁**：医院 / 康复诊所
- **单价 ~$150-250k**，附加培训 + 服务合同
- **临床数据驱动**：与医院联合发表临床证据是关键销售引擎

## 六、与同行对比

| 维度 | Ekso EksoNR | ReWalk Personal 6.0 | Cyberdyne HAL |
|---|---|---|---|
| 主市场 | **医院诊所** | 患者家用 | 医院 + 老龄护理 |
| 适应症 | **中风 + SCI 急性** | 完全 SCI 截瘫 | 多元（含 ALS / 肌营养不良） |
| 控制 | 力 + 姿态 | 重心 + 拐杖 | EMG（BES） |
| 价格 | $150-250k（机构）| ~$80-100k（个体）| 租赁为主 |
| 患者数据规模 | 数千例临床记录 | 千级 | 数千 + 老龄护理 |

## 七、竞争与壁垒

- **直接对手**：ReWalk · Cyberdyne · Wandercraft · 国产玩家（傅利叶 ExoMotus / 程天 / 大艾）
- **壁垒**：
  - **临床数据池**（中风急性期康复证据）—— 这是医院采购的硬通货
  - **Berkeley 工程基因**（Kazerooni 教授至今学术活跃）
  - **B2B 销售渠道 + 培训体系**
- **弱点**：
  - 财务多年亏损 / 市值长期低位
  - 中国玩家（傅利叶 / 程天 / 大艾）以 1/3 价格抢中端市场
  - 工业 EksoVest 路线已基本剥离，专注医疗反而限制规模

## 八、关键风险

- **现金流压力**：长期亏损经营
- **中国低价竞争**侵蚀全球中低端机构市场
- **PT/OT 治疗师培训瓶颈** —— 医院买了机器但无人会用是真痛点

## 九、我的判断

> 1. **Ekso 是医疗外骨骼"B2B 医院诊所" 唯一稳健代表** —— 不像 ReWalk 那样赌患者家用市场（基本未跑通）
> 2. **2026-2028 中国低价玩家是最大威胁**——傅利叶 ExoMotus + 程天 在欧洲 / 中东以 1/3 价格已经有进展
> 3. **被并购概率不低**——市值 < $100M 时是 Medtronic / Stryker 等大医械潜在并购对象
> 4. **EksoUE 上肢方向**有差异化机会，但中风上肢康复市场比下肢小
>
> **我可能错在哪里**：（a）AI 辅助治疗师培训突破，让医院使用率 3-5x，市场打开；（b）中国玩家受地缘 / 监管限制无法大规模出海，Ekso 守住欧美机构市场。
