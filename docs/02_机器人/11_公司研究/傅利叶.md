# 傅利叶（Fourier Intelligence）

> 最后更新：2026-04-24
>
> 傅利叶是 **上海起家的机器人 / 康复医疗公司**——2015 年顾捷创立，**早期做康复机器人（外骨骼）**，2023 转型做人形。2024 发布 **GR-1 / GR-2**，主打 **通用双足 + 开放生态**。2024 B+ 轮估值 ~$500M+。

## 一句话定位

傅利叶 = **"从康复走向通用" 的人形公司** —— 康复医疗 10 年积累的 **骨骼运动学 + 柔顺控制**是差异化，产品向**科研 + 工业**扩展。

## 一、公司速览

- **创立**：2015
- **总部**：上海
- **创始人 / CEO**：顾捷
- **员工**：~300
- **估值**：**~$500M+**（2024-12 B+ 轮）
- **投资方**：沙特 VCOM、软银、翎盛资本、浦东创投

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2015 | 成立，主攻康复外骨骼 |
| 2019 | Fourier M2（康复）|
| 2023-07 | **GR-1 首发** |
| 2024-08 | GR-2 发布 |
| 2024-12 | B+ 轮 $120M+ |
| 2025 | GR-3 + 出海 |

## 三、业务与产品

- **GR-1 / 2**：通用人形双足
- **ExoMotus**：康复外骨骼（核心现金流）
- **科研市场**：清华 / 上交等实验室采购
- **2025 开始工业试点**

## 四、技术路线

- **运动学 + 柔顺控制**（康复基因）
- 2024 加强 VLA 研究
- 开放 API + 开发者生态

## 五、商业模式

- 康复医疗 ¥200-500M/年（现金牛）
- 人形早期销售 ¥50-100M
- 科研 + 工业混合

## 六、竞争与壁垒

- 对标：智元、宇树、优必选
- 壁垒：**康复 10 年积累 + 中东沙特订单**
- 弱点：品牌声量低于稚晖君派

## 七、关键风险

- 康复业务萎缩（医保变化）
- 人形工业落后智元 / 宇树
- 创始人战略分散

## 八、我的判断

> 1. **傅利叶是"最有医疗基因的人形公司"** —— 康复外骨骼是独特护城河
> 2. **不会是头部赢家** 但**会有稳定 5-10% 份额**
> 3. **沙特 / 中东市场**是隐藏优势
>
> **我可能错在哪里**：康复外骨骼 + 人形协同做出老龄化产品爆款。
