# Roam Robotics

> 最后更新：2026-04-29
>
> Roam 是 **美国旧金山 2014 创立的软体外骨骼公司**——**Tim Swift** 创立。**软体气动外骨骼**路线代表玩家——区别于 ReWalk / Cyberdyne 的金属刚性结构。**早期主攻消费滑雪辅助** → 战略调整后转向**膝关节康复 + 户外辅助**。**消费 / 准消费段外骨骼里融资规模较大**的玩家。**Arc'teryx 等户外品牌合作背景**让 Roam 在消费段有差异化。

## 一句话定位

Roam = **"消费外骨骼软体派"** —— 不做截瘫患者重度康复（那是 ReWalk / Wandercraft 的战场），也不做工业重负载（Sarcos / Ottobock 的战场），而是**膝关节辅助 + 户外运动 + 老龄助步**这个**中间地带**。**软体气动 + 轻量 + 低价**是技术差异化，但**消费市场至 2025 仍未跑通规模化**。

## 一、公司速览

- **创立**：2014
- **总部**：San Francisco, California（亦在 Carlsbad 等地有办公）
- **创始人 / CEO**：**Tim Swift**
- **员工**：~30-50（具体未公开）
- **融资**：累计 ~$50M+（多轮 A / B + 部分债务融资）
- **估值**：未公开披露
- **官网**：roamrobotics.com

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2014 | 公司成立（前身 OtherLab 衍生） |
| 2018-2019 | **Ascend** 滑雪辅助外骨骼（消费款）公开演示 |
| 2020-2021 | 消费滑雪市场推广（与 Arc'teryx 等户外品牌合作） |
| 2022-2023 | **战略调整**：弱化纯消费滑雪，转向**膝关节康复 + 户外通用辅助** |
| 2024-2025 | 与户外品牌 + 康复诊所渠道双线并行 |

## 三、业务与产品

- **Ascend / Skip**（消费款）：滑雪膝关节辅助
- **Knee 康复款**：术后 / 老年膝关节问题辅助
- **户外辅助**：登山 / 长途徒步增强
- **Arc'teryx Skip 合作款**（2024 公布）：高端户外品牌合作

## 四、技术路线（独家差异化）

- **气动 + 软体织物**结构（不是金属刚性 / 电机驱动）
  - **优势**：极轻（< 5 lbs）、易穿戴、低价
  - **劣势**：力量上限低（不适合截瘫支撑）、需便携气源 / 电池
- **专利组合**：软体气动驱动 + 织物-人体接口
- **OtherLab 工程基因**（Saul Griffith 等湾区软体机器人研究网络）

## 五、商业模式

- **B2C 消费**（滑雪 / 户外） + **B2B 康复诊所**（膝关节）双线
- **品牌合作渠道**（Arc'teryx）—— 是消费外骨骼最有效的进入渠道
- **价格区间**：~$1k-3k / 件（远低于医疗外骨骼 $50k+）

## 七、竞争与壁垒

- **直接对手**：Honda Walking Assist · Wandercraft Eve（未来）· Skip（兄弟公司或子品牌）
- **壁垒**：
  - **软体气动专利** + 美国西海岸软体机器人生态
  - **Arc'teryx 等品牌合作** 渠道
  - **价格 / 重量优势**对消费市场友好
- **弱点**：
  - **多次战略 pivot** 让公司聚焦受影响（滑雪 → 膝盖康复 → 户外）
  - **气源 / 电池续航**约束消费场景
  - **市场尚未跑通**：消费外骨骼整体市场 < $100M / 年

## 八、关键风险

- **消费市场永不规模化** —— 这是软体外骨骼最大风险
- **大公司 / 户外品牌自研**：Arc'teryx 等品牌可能内化技术
- **医疗 / 康复路径监管慢**：膝关节康复市场进入需要 FDA 二类批准

## 九、我的判断

> 1. **Roam 是消费外骨骼"软体派"代表**——技术路径不同于 90% 行业玩家，差异化清晰
> 2. **2026-2028 关键看 Arc'teryx 合作款规模**——消费款外骨骼能否跑通的真正测试
> 3. **5 年内消费外骨骼整体市场仍是 niche** —— 不会有 iPhone 时刻
> 4. **被并购概率较高**——户外品牌 / 大型医疗器械公司可能收购
>
> **我可能错在哪里**：（a）滑雪 / 户外运动外骨骼意外爆款（35-55 岁高净值人群），消费段被点燃；（b）AI 辅助步态分析与软体外骨骼结合，催生新一波 fitness 应用。
