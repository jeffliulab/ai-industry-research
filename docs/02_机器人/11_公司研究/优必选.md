# 优必选（UBTECH）

> 最后更新：2026-04-24
>
> 优必选是 **中国机器人行业老牌玩家**——2012 年周剑创立，**2023-12 港交所 IPO**（09880.HK），成为"人形机器人第一股"。旗舰产品 **Walker X**（2021）+ **Walker S1**（2024）面向工业场景，2024-11 **Walker S1 进入比亚迪 / 奥迪工厂试点**。

## 一句话定位

优必选 = **"中国人形机器人上市第一股"** —— 12 年老牌 + 教育机器人起家（曾量产数百万台 Alpha / Jimu），**2023 IPO 后重心转向人形工业**。**规模 + 渠道有优势，但 AI 大脑落后智元**。

## 一、公司速览

- **创立**：2012
- **总部**：深圳
- **创始人 / CEO**：周剑
- **员工**：~1500
- **上市**：**2023-12-29 港交所 IPO**（09880.HK）
- **市值**：~HK$30-50B 波动（2025）
- **投资方（上市前）**：腾讯、启明创投、科大讯飞、鼎晖

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2012 | 成立 |
| 2016 | Alpha 教育机器人（爆款）|
| 2018 | 春晚舞蹈表演 |
| 2021 | **Walker X 发布** |
| **2023-12** | **港股 IPO** |
| 2024-05 | Walker S1 发布 |
| 2024-11 | **进入比亚迪 / 奥迪工厂** |
| 2025 | Walker S2 研发 + 量产扩大 |

## 三、业务与产品

- **Walker 系列**：工业人形（S1 / S2）
- **Alpha / Jimu**：教育机器人（历史业务）
- **警卫 / 服务机器人**：多样化产品
- **客户**：比亚迪、奥迪、富士康（试点）

## 四、技术路线

- 教育机器人背景 → 工业人形
- AI 大脑相对保守（非 VLA-first）
- 2025 开始追赶端到端

## 五、商业模式

- 硬件销售为主
- 2024 营收 ¥1B+（教育 + 工业混合）
- 工业人形占比逐步上升

## 六、竞争与壁垒

- 对标：智元、宇树、特斯拉 Optimus
- 壁垒：**12 年技术积累 + 港股上市融资 + 渠道**
- 弱点：AI 大脑追赶、盈利能力弱

## 七、关键风险

- 智元 / 宇树高速崛起抢订单
- 港股估值下行
- 教育机器人业务萎缩

## 八、我的判断

> 1. **优必选是"传统优势 vs AI 后发" 的典型矛盾**
> 2. **上市融资 + 老牌 brand** 让其不会快速失败
> 3. **人形工业赛道追赶智元** 需 2-3 年
> 4. **大概率成为"中国第二梯队" 人形公司**
>
> **我可能错在哪里**：比亚迪 / 奥迪合作放大后优必选成为大客户首选。
