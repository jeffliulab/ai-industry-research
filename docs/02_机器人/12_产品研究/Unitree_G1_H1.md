# Unitree G1 / H1

> 最后更新：2026-04-24
>
> Unitree（宇树科技）是 **中国四足机器人全球龙头 + 最便宜人形玩家**——**H1**（2023 发布，1.8m 全尺寸，$90k+）+ **G1**（2024 发布，1.3m 教育 / 研究版，$16k 起）是爆款。**2024-02 G1 一开卖全球断货**，"**便宜得离谱**"的人形让 Unitree 成为研究圈标准硬件。

## 一、产品定位

Unitree G1 = **"人形版 Raspberry Pi"** —— $16,000 的全尺寸双足，**开源 SDK + 高校 / 科研实验室批量采购**。H1 是旗舰高性能版。**让人形从"企业玩具"变"人人能买"**。

## 二、核心能力与架构

### H1（旗舰）
- 1.8m、47kg
- 走 1.5m/s
- $90,000+

### G1（爆款）
- 1.3m、35kg
- 走 2m/s
- **$16,000 起**（基础版）
- 23 DOF 手臂 + 灵巧手

### 技术栈
- 自研电机 + 减速器
- **开源 SDK** 吸引开发者
- 数据集开源（G1 datasets）

## 三、版本与路线图

## 四、定价与商业化

- **G1**：$16,000-$20,000
- **H1**：$90,000+
- 2024-2025 累计销售数千台
- 2025 ARR 估算 ¥300M+

## 五、用户反馈

- **"人形研究的新标准硬件"** —— 全球高校批量采购
- **价格屠夫**：2-5x 便宜于美国对手
- **开源 SDK + 社区** 活跃
- 批评：**手部精度 / 负载不如 Figure**
