# 机器人

> **传统工业 / 协作 / 服务机器人 + 外骨骼 + 机器人产业链**。本板块视角是**硬件 + 供应链**，不含人形机器人本体（见 [人形机器人板块](../02_人形机器人/index.md)）；不含具身大脑（见 [具身智能板块](../03_具身智能/index.md)）。

## 板块定位

人形机器人 2024-2026 爆发后，"机器人"这个名词被狭义理解为人形。但**传统工业 / 协作 / 服务机器人**仍是 ~$50B 全球市场的存量基本盘；**外骨骼**是新兴的"穿戴式机器人"第五类。**核心零部件供应链**（减速器 / 丝杠 / 电机）是上述所有机器人形态共享的硬件底座——也是人形板块绕不开的话题。
