# 人形机器人 2026 年度预测

> 最后更新：2026-04-30
>
> 仿 [Benaich State of AI](https://www.stateof.ai/) 体例做"年度可证伪预测 + 自评打分"——人形机器人板块的**12 条预测**，覆盖技术 / 商业 / 资本 / 地缘四个维度。**所有预测附观察指标 + 打分日期**。下一年（2027-Q1）回来打分。

## 预测原则

按 "行业研究标准方法" 第 2 节"前瞻写法的纪律"：
- **可证伪**（明确 yes/no 标准）
- **附观察指标**（如何判断成真）
- **附打分日期**（何时回来复盘）
- 预测**给概率**而非确定性陈述

## 12 条 2026 年度预测

### 一、技术 / 产品

**1.** **Tesla Optimus 2026 内会有 ≥10,000 台投入 Tesla 自有工厂使用**（**60%** 概率）
- 观察：Musk / Tesla 财报披露
- 打分：2027-01

**2.** **Figure 03 灵巧手能力公开演示中能完成"端茶倒水级"5 步以上序列**（**70%** 概率）
- 观察：Figure 官方视频
- 打分：2026-Q4

**3.** **至少一家中国玩家（宇树 G2 或 智元 X2）将整机价格压到 $5-8k**（**75%** 概率）
- 观察：官方报价
- 打分：2026-Q4

**4.** **NVIDIA GR00T 不会成为 Android of robotics**——主要玩家（Tesla / Figure / 宇树）保持自研大脑（**80%** 概率）
- 观察：主要厂商公开声明 / 产品采用
- 打分：2027-01

### 二、商业化

**5.** **Agility Digit 2026 出货 ≥3,000 台**（**55%** 概率，已有 Amazon / GXO 部署但规模未公开）
- 观察：公司公告 + 客户披露
- 打分：2027-01

**6.** **欧美至少 2 家工厂将"人形小批量量产部署"作为 PR 核心**（**80%** 概率）
- 观察：BMW / Mercedes / Foxconn 等公开案例
- 打分：2026-Q4

**7.** **2026 全球人形出货 < 200,000 台**（**85%** 概率，远低于高盛 2030 1-2M 预测）
- 观察：IFR / 摩根士丹利年度统计
- 打分：2027-Q1

### 三、资本 / 估值

**8.** **Figure 估值 2026 年内不会突破 $80B**（**75%** 概率，但 IPO 仍在公开窗口外）
- 观察：私募融资公告
- 打分：2027-01

**9.** **2026 内至少有 1 家美国 / 欧洲人形玩家被并购或宣布关闭**（**60%** 概率）
- 观察：公开新闻
- 打分：2027-01

**10.** **宇树或智元至少一家在 2026-2027 完成 IPO 或正式提交招股书**（**50%** 概率）
- 观察：港交所 / 上交所披露
- 打分：2027-Q2

### 四、地缘 / 政策

**11.** **美国不会出台专门针对中国人形机器人的全面进口管制**（**85%** 概率）
- 观察：BIS / USTR 公告
- 打分：2027-01

**12.** **中国"具身智能 + 人形机器人"会被列为"十五五"规划重点**（**75%** 概率）
- 观察：2026-2027 中央经济工作会议 / 五年规划文件
- 打分：2027-Q2

## 自评打分规则

参考 Benaich State of AI 体例：
- **完全成真**：1 分
- **大部分成真**（要点中 ≥70% 成立）：0.5 分
- **未成真**：0 分
- **过度乐观 / 悲观**也单独标记，不算分

总分 / 12 是当年预测命中率。**目标：60% 以上命中率**——低于这个值说明前瞻能力还要提升。

## 重大不确定性

以下事件**显著影响以上预测**，但本身概率难评：
- Tesla / Musk 突然战略变更（人形非优先）
- 中美科技战升级到具身机器人 / 人形机器人
- 某家公司发生重大事故（伤人 / 损坏）触发监管
- 新兴技术突破（如量子触觉 / 生物启发驱动）改变工程范式

## 上一年（2025）回顾占位
