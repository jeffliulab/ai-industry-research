# 脑机接口

> 脑机接口（Brain-Computer Interface, BCI）是 **"人脑 ↔ 计算机" 的直接通路** —— 绕过肌肉和感官。从医疗（瘫痪恢复、视觉、神经假体）到消费（注意力监测、AR 控制）再到增强（记忆、注意力）的全光谱产业。2024-2026 是 BCI 从实验室走向商业化的拐点。

## 为什么独立成板块

- Stanford HAI 与 MIT Technology Review 都把 neurotechnology 视为独立方向
- **独立玩家群**：Neuralink、Synchron、Precision Neuroscience、Paradromics、Blackrock Neurotech、BrainGate、Kernel、Onward、脑虎、BrainCo —— 10+ 家专注做这一方向
- **与 AI 医疗的边界**：AI 医疗偏**软件 + 药物**（AlphaFold / Abridge），BCI 偏**硬件 + 神经信号**，清晰可分
- **与机器人 / 具身的边界**：不是移动能力，是**人机接口**

## 主要玩家

**侵入式（高带宽）**：
- Neuralink（Musk，2024 首例人体试验）
- Synchron（血管内支架电极，FDA IDE 已批）
- Precision Neuroscience（薄膜电极，2024 首例人体植入）
- Paradromics（Connexus）
- Blackrock Neurotech（老牌，学术研究主力）
- BrainGate（多机构学术联盟）

**非侵入 / 消费**：
- Kernel（脑成像头盔）
- Onward Medical（脊髓损伤神经刺激）
- BrainCo 强脑科技（中国，Focus 头环）

**中国**：
- 脑虎科技（侵入式，清华系）
- BrainCo（消费级 + 假肢）

## 代表产品

- Neuralink N1（可植入无线 BCI）
- Synchron Stentrode（血管内 BCI）
- Precision NSX（薄膜电极）
- BrainGate 系统（学术旗舰）
- BrainCo Focus（消费级注意力头环）

!!! warning "极强监管约束"
    BCI 是最受监管约束的 AI 相关产业之一 —— **FDA / NMPA 医疗器械审批 + 神经数据伦理 + 人类尊严边界**。2024 年加州通过全美第一个"神经权利法案"（SB 1223），将脑数据纳入隐私保护。本站内容为公开信息整理，**不构成任何医学或投资建议**。
