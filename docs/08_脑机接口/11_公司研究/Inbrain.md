# Inbrain Neuroelectronics

> 最后更新：2026-04-29
>
> Inbrain 是 **西班牙巴塞罗那出身的石墨烯（graphene）BCI 公司**——2020 拆分自 Catalan Institute of Nanoscience and Nanotechnology（ICN2）。**唯一商业化使用石墨烯电极**——这是全行业对"长期生物相容 + 高带宽"诉求的另一条材料路径，与 Neuralink Thread / 脑虎柔性电极、Utah Array 硅基形成对比。**与 Merck KGaA / Cumberland Pharma 合作，主攻 Parkinson's + 神经肿瘤**。

## 一句话定位

Inbrain = **"石墨烯 BCI 的全球领先者"** —— 用单层石墨烯做电极阵列，**生物相容性 + 信号信噪比理论领先**。**临床切入点是 DBS 替代**（帕金森） + **神经肿瘤手术辅助**（脑胶质瘤切除时的神经监测）—— 是欧洲 BCI 旗手企业。

## 一、公司速览

- **创立**：2020（拆分自 ICN2 + Universitat Autònoma de Barcelona）
- **总部**：Barcelona, Spain
- **创始人 / CEO**：Carolina Aguilar（CEO）+ Kostas Kostarelos（CSO，巴塞罗那 / 曼彻斯特石墨烯研究领头人）
- **员工**：~50（具体数字未公开）
- **融资**：累计 ~€60M+（据公开报道，C 轮 + Merck KGaA 战略投资）
- **估值**：未公开

> ⚠ 数字以官方公告为准。

| 时间 | 里程碑 |

## 三、产品 / 业务

- **石墨烯电极阵列**：核心技术
  - 单层 / 少层石墨烯做导电层 + 柔性聚合物基底
  - **理论优势**：
    - 信噪比高（电荷注入容量大于铂、铱氧化物等传统电极材料 10-100x）
    - 生物相容性好（石墨烯本身 + 柔性基底降低慢性炎症）
    - **MRI 兼容性**（不发热、不伪影）—— DBS 用户能继续做 MRI 是临床关键
- **两条临床管线**：
  1. **帕金森 DBS 替代**：石墨烯电极版深部脑刺激（更高分辨率 + MRI 兼容）
  2. **神经肿瘤术中辅助**：脑胶质瘤切除时实时神经监测——避免运动 / 语言皮层损伤
- **学术输出**：与 Manchester Graphene Institute 联合实验

## 四、技术路线

- **石墨烯 vs 硅 vs 柔性聚合物 vs 金属电极**：本质是材料创新派
  - 不是新的"信号读取范式"——还是 ECoG / DBS 类信号
  - 但电极性能（信噪比 + 寿命 + 生物相容）的提升是**横向赋能** —— 任何 BCI 公司都可以受益
- **已发表 Nature Nanotechnology / Nature Biomedical Engineering 等**多篇高质量论文
- **MRI 兼容性**是工程关键：传统金属电极在 MRI 下会发热 / 伪影，石墨烯可避免——决定 DBS 患者是否能做长期影像随访

## 五、与同行对比

| 维度 | Inbrain | Neuralink | 脑虎 | Blackrock Utah Array |
|---|---|---|---|---|
| 电极材料 | **石墨烯** | 铂铱（柔性 Thread） | 柔性导电聚合物 | 硅基 + 铂尖 |
| 路径 | DBS 替代 + 术中监测 | 皮层穿透 BCI | 皮层穿透 BCI | 皮层穿透电极阵列 |
| MRI 兼容 | **是**（关键差异化）| 待验证 | 待验证 | 否（伪影）|
| 商业化路径 | **B2B**（与 Merck 合作）| **B2C**（医疗器械）| 自营医疗器械 | B2B 学术供应商 |
| 监管区域 | 欧洲 CE / FDA | FDA | NMPA | FDA / 国际 |

**判读**：Inbrain 不是 Neuralink 的直接对手——它是**给整个 BCI 产业供应新一代电极平台**的"卖铲人"路径，类似 ARM 在芯片产业。

## 六、商业模式

- 未大规模商业化
- 短期：与 Merck KGaA 等大型药企 / 医疗器械巨头联合开发（B2B 平台型）
- 长期：自有 DBS 替代器械（B2C 医疗器械）+ 石墨烯电极授权 / 供应

## 七、竞争与壁垒

- **直接对手**：Neuropace（皮层电极）+ Medtronic / Boston Scientific（DBS）+ 各家 BCI 公司的电极工艺
- **壁垒**：
  - **石墨烯专利组合 + Manchester / ICN2 学术深度**（[7 Powers · Process Power](../../99_方法论/框架_7_Powers.md)）
  - Merck KGaA 战略合作（资本 + 渠道）
  - **欧洲监管合规优势**（在欧洲 CE 是先发市场）
- **弱点**：
  - 不是 BCI **应用层创新**——而是材料层；商业模式比 Neuralink 慢
  - 美国市场远 + FDA 审批要重新做
  - 技术差异化"信号质量"在临床终点是否真有可观察差别——需大样本验证

## 八、关键风险

- **临床差异化失败**：石墨烯电极在临床效果上如果与传统电极无明显差距，长期估值天花板低
- **大药企并购被边缘化**：Merck 战略投资可能演变成被收购，公司独立路径终止
- **资本环境**：欧洲 BCI 资本远小于美国

## 九、我的判断

> 1. **Inbrain 是 BCI 产业的"卖铲人"赌局**——若石墨烯电极成主流，Inbrain 是平台供应商；若不成，公司估值上限有限
> 2. **MRI 兼容性是被低估的临床卖点** —— 帕金森 DBS 患者长期需要影像随访，是真痛点
> 3. **欧洲 BCI 旗手代表**——和美国 / 中国格局形成第三极
> 4. **2026-2028 关键看 DBS 替代临床试验疗效数据**
>
> **我可能错在哪里**：（a）传统铂铱电极工艺持续优化，石墨烯优势缩小；（b）柔性聚合物（Neuralink Thread / 脑虎）才是长期赢家，石墨烯刚性 + 柔性基底组合在 5+ 年体内稳定性上不及预期。
