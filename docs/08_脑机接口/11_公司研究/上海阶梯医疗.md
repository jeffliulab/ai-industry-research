# 上海阶梯医疗（StairMed）

> 最后更新：2026-04-29
>
> 阶梯医疗是 **中国半侵入式 BCI 代表**——上海起家，与**清华大学 洪波** 团队深度绑定。**NEO（Neural Electronic Opportunity）系统**：颅骨内、硬膜外贴附式 ECoG 电极，**不穿透皮层、无线供电**。**2023 完成首例人体植入**（首都医科大学宣武医院 赵国光 团队），**2024 多例**。与脑虎"全侵入柔性电极"路线**互补而非竞争**——构成中国 BCI 双轨格局。

## 一句话定位

阶梯医疗 = **"中国半侵入 BCI 第一**" —— 用 ECoG 在 **侵入式精度** 与 **非侵入式安全** 之间取**轻量化中点**，类比 Synchron Stentrode 的"无开颅穿透"思路。**首例人体患者 2023 已能用脑电控制鼠标、操作微信**。

## 三、产品 / 业务

- **NEO 系统**（Neural Electronic Opportunity）：核心产品
  - **颅内、硬膜外** 电极阵列（贴附在硬脑膜上方）
  - **无线供电 + 无线通讯**（体外感应线圈）
  - **不穿透皮层** —— 与 Neuralink Thread / 脑虎柔性电极的**穿透式**形成对比
  - **配套气动手套**（外部执行器）：让患者实现抓握等基本动作
- **临床应用**：高位截瘫 → 中风 → ALS 顺序拓展（参考 BrainGate / Synchron 路径）

## 四、技术路线

- **半侵入 ECoG（Electrocorticography）**：电极位于硬膜外
  - **侵入度**：要钻颅骨 + 切开头皮，但**不刺破硬膜、不接触皮层**
  - **信号质量**：好于头皮 EEG，弱于 Utah Array / Thread 单神经元
  - **生物相容性 + 长期稳定性**：理论优于穿透式（无慢性炎症 / 神经胶质瘢痕）
- **无线 + 无电池体内方案**：**体外线圈感应供电**，体内无电池（避免长期手术更换）
- **AI 解码**：与清华团队联合研发；侧重**运动意图解码**（鼠标 / 抓握指令）

## 五、与脑虎 / Neuralink / Synchron 路线对比

| 维度 | StairMed NEO | 脑虎 | Neuralink N1 | Synchron Stentrode |
|---|---|---|---|---|
| 侵入度 | **半侵入（硬膜外 ECoG）** | 全侵入（皮层穿透柔性电极） | 全侵入（Thread） | 血管内（颈静脉支架电极） |
| 通道数（公开） | 数十至数百级 | ~1000+ | 1024 | 数十级 |
| 信号源 | ECoG（局部场电位） | 单神经元 + LFP | 单神经元 | 血管壁附近 LFP |
| 手术 | 开颅 + 硬膜外贴附 | 开颅 + 机器人植入 | 开颅 + R1 机器人 | **不开颅**（介入手术） |
| 体内电池 | **无**（无线供电） | 有 | 有 | 有 |
| 首例人体 | 2023-10（中国） | 2024（中国） | 2024-01（美国） | 2021-2022（美国） |
| 监管 | NMPA | NMPA | FDA | FDA |
| 适应症 | 截瘫起步 | 截瘫 / ALS 起步 | 截瘫起步 | 截瘫起步 |

**判读**：NEO 是与 Synchron 同类的"侵入轻量化"路径，但落点在**颅内 + 硬膜外**而非**血管内**——本质是 ECoG 学派的工程化产品，不是 Neuralink / 脑虎的同台竞争者。

## 六、商业模式

- **未商业化**（处临床试验阶段）
- 商业化路径预期：医疗器械 + per-procedure 收费 + 配套耗材 / 软件
- **中国 NMPA 三类医疗器械** 是核心准入门槛

## 七、竞争与壁垒

- **直接对比**：脑虎（侵入路径）、Synchron（血管内）、BrainGate（学术 ECoG）
- **壁垒**：
  - 清华洪波团队 **十年 ECoG 学术积累**（[7 Powers · Process Power](../../99_方法论/框架_7_Powers.md)）
  - 宣武医院 神外临床合作 + 中国患者池
  - **"硬膜外 + 无线 + 无电池"工程组合**——是有差异化的中国本土方案
- **弱点**：
  - **通道数远低于 Neuralink / 脑虎** —— 上限决定能做的复杂度
  - 公司体量小、融资节奏慢
  - 商业化路径长（NMPA 三类器械审批通常 3-5 年）

## 八、关键风险

- **临床并发症**：硬膜外感染 / 出血等是开颅手术固有风险
- **技术天花板**：ECoG 解码精度上限是否够支撑"代替键盘 / 灵巧抓握"
- **资本环境**：中国 BCI 融资集中度高（脑虎吸走多数注意力），StairMed 二线身位
- **政策**：NMPA 对侵入式 BCI 监管标准仍在制定，进度可能慢于预期

## 九、我的判断

> 1. **StairMed 是中国 BCI 双轨格局的"另一极"**——脑虎走 Neuralink 路线（重侵入 + 高带宽），StairMed 走 Synchron 路线（轻侵入 + 中带宽）。**两条路都需要被走过**——不是零和关系
> 2. **2026-2028 关键看 NEO 临床试验扩展速度**：能否累积到 **10+ 例人体**且无重大并发症，决定 NMPA 路径
> 3. **真正的差异化护城河是"无线 + 无电池"工程化**——如果能稳定 5 年以上不更换，对患者吸引力会大于 Neuralink Thread
> 4. **5 年内仍是医疗 BCI**——消费级、增强级与 StairMed 路线无关
>
> **我可能错在哪里**：（a）若 ECoG 解码上限被证明远不足以支撑实用，则 StairMed 路线被边缘化；（b）若脑虎大规模融资 + 监管绿灯，StairMed 在中国市场份额被挤压。
