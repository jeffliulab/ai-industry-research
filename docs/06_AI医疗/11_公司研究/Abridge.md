# Abridge

> 最后更新：2026-04-24
>
> Abridge 是 **临床 Copilot 领跑者**——2018 由医生 Shiv Rao 创立于 Pittsburgh。**"AI 监听医患对话 → 自动生成病历"**。2025-02 D 轮 **$250M，估值 $2.7B**。Kaiser、Emory、Yale、Cleveland Clinic 等顶级医院系统签约。2025 ARR 估算 **$100-200M+**，增速 5-10x 年增长。

## 一句话定位

Abridge = **"AI 医疗最干净的商业化故事"** —— 产品好、ROI 清晰、Big Health System 批量采购、CEO 是医生。**AI 医疗"Cursor 时刻"代表**。

## 一、公司速览

- **创立**：2018
- **总部**：Pittsburgh, PA
- **CEO**：Shiv Rao（心脏病学医生）
- **员工**：~300
- **估值**：**$2.7B**（2025-02 D 轮 $250M）
- **投资方**：Spark Capital、Bessemer、Google Ventures、USV、Lightspeed

## 二、历史沿革

| 时间 | 里程碑 |
|---|---|
| 2018 | 成立 |
| 2023 | 2000+ 医生用户 |
| **2024-02** | **Kaiser 合作** 扩大 |
| 2024-10 | C 轮 $150M |
| **2025-02** | **D 轮 $250M，估值 $2.7B** |
| 2025 | Emory / Yale / Cleveland 扩展 |

## 三、技术路线

- **ASR + LLM + 医学实体识别**
- **Epic / Cerner EMR 深度集成**
- **HIPAA 合规 + BAA**
- **定期 fine-tune on 医生反馈**

## 四、商业模式

- Per-provider / Per-encounter
- 2025 ARR ~$150M（估）
- 毛利 60%+
- **扩张速度快，但毛利被基础模型成本侵蚀**

## 五、竞争与壁垒

- 对标：Microsoft Dragon Ambient / Nuance、Epic Ambient、Suki
- 壁垒：**医生 CEO brand + 顶级医院 logo + EMR 深度**
- 弱点：**Epic 免费策略** 是最大风险

## 六、我的判断

> 1. **Abridge 是 AI 医疗 2024-2026 最大赢家**
> 2. **2026-2027 IPO** 估值 $5-10B 可能
> 3. **Shiv Rao 是 AI 医疗最被低估 CEO**
> 4. **Epic Ambient 是真正威胁**
>
> **我可能错在哪里**：Epic 免费策略让 Abridge 增速骤降。
