# Recursion Pharmaceuticals

> 最后更新：2026-04-24
>
> Recursion（$RXRX，2013 成立）是 **AI 药物发现上市公司代表**——盐湖城起家，**表型筛选（Phenomics）+ ML**路线。2021 IPO，2023-08 **NVIDIA 战略投资 $50M**，2024-08 **收购 Exscientia $688M**（同业合并）。股价波动大但仍是行业标杆。

## 一句话定位

Recursion = **"AI 药物发现上市第一股"** —— **表型数据 + 机器学习 + 真机 lab 一体化**，与 Insilico / Isomorphic 等"结构派" 不同，走 **数据驱动表型路线**。

- **创立**：2013
- **总部**：Salt Lake City, Utah
- **CEO**：Chris Gibson
- **员工**：~800（并 Exscientia 后）

## 三、技术路线

- **表型筛选（Phenomix）+ ML**
- **细胞图像数据集 万亿像素**
- **BioHive-2 超算**（NVIDIA 赞助）

## 四、Pipeline

- **REC-617**（BRD4 抑制剂）Phase 1/2
- **REC-994**（血管畸形）Phase 2
- **与 Roche / Bayer / Sanofi / Merck 合作**

## 五、商业模式

- **自有管线 + 药厂合作**
- 2024 营收 ~$80M（主要合作金）
- **亏损** （烧钱研发）

## 六、竞争与壁垒

- 对标：Isomorphic、Insilico、Genesis Therapeutics
- 壁垒：**表型数据独特 + NVIDIA 合作**
- 弱点：**无药物批准** + 股价压力
