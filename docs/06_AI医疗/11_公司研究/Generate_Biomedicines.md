# Generate Biomedicines

> 最后更新：2026-04-24
>
> Generate Biomedicines（Flagship Pioneering 孵化，2018 创立）是 **Generative AI 蛋白设计 + 药物公司**——2023-09 **C 轮 $273M，估值 $1.7B**。**Chroma 扩散模型 + RFdiffusion-style** 技术，**与 Amgen 签 $1.9B deal**（2022），是 AI 蛋白设计最大商业化公司之一。

## 一句话定位

Generate = **"蛋白 AI 设计商业化代表"** —— Flagship（Moderna 母公司）出品光环 + 大型 Pharma 合作 + AI 平台规模化。**不上市**但**估值 $1.7B**。

## 一、公司速览

- **创立**：2018（Flagship Pioneering 孵化）
- **总部**：Somerville, MA
- **CEO**：Alex Kiselyov
- **员工**：~400
- **估值**：**$1.7B**（2023-09 C 轮 $273M）

## 三、技术路线

- **Chroma 扩散模型**（蛋白结构 + 序列联合）
- **多 target 类型**：antibody、peptide、enzyme
- **Lab-in-the-loop**：AI + 真机实验闭环

## 四、合作 / Pipeline

- **Amgen**（$1.9B，2022）
- 多个内部管线 Phase 1
- 其他药厂合作

## 五、竞争与壁垒

- 壁垒：**Flagship 关系 + Amgen 大单**
- 弱点：**未上市** 流动性
