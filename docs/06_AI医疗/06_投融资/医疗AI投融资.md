# 医疗 AI 投融资全景

> 最后更新：2026-04-30
>
> AI 医疗 2024-2026 是**药物发现资本回流 + 临床 Copilot 爆发**的双重热潮。本文盘点全球玩家估值梯队 + 资本流向 + 退出路径。**药物发现派**经历 2022-2023 估值修正后回暖；**临床 Copilot 派**（Abridge / Nabla / Hippocratic）2024 多次大额融资。

## 一句话结论

医疗 AI 2026 投融资格局：**Isomorphic Labs（Alphabet 系）+ Recursion / Insilico / Absci 等药物发现系**+ **Abridge / Nabla / Hippocratic 等临床 Copilot 系**+ **Tempus / Caris / 推想 等影像诊断系**。**临床 Copilot 是当前估值最贵的细分**——Abridge $5B+ 估值、Hippocratic 等多家融资 $200M+。

## 三条关键要点

1. **临床 Copilot 估值飙升**：Abridge、Nabla、Hippocratic 等 2024-2025 多家完成 D 轮 / E 轮，单家估值 $1-5B
2. **药物发现派复苏但分化**：Recursion 整合 Exscientia（2024 合并）+ Isomorphic Labs 仍嵌入 Alphabet
3. **中国梯队规模较小**：推想 / 深透 / 鹰瞳 等多家上市但市值多 < $1B

## 资本流向变化

```
2014-2018  早期：Recursion / Insilico / Atomwise 起步
2019-2021  狂热期：Recursion IPO（2021，估值 $7B+）+ 多家药物发现 D-E 轮
2022       拐点：药物发现派估值修正 50-70%（"AI 加速 ≠ 药物成功"）
2023       药物发现继续修正；临床 Copilot 兴起
2024       临床 Copilot 爆发：Abridge / Nabla / Hippocratic 大额融资
2025       药物发现复苏（Recursion + Exscientia 合并）+ 临床 Copilot 估值再升
2026       格局清晰：临床 Copilot > 药物发现 > 影像诊断 三类各有头部
```

## 资本结构

### 美国
- **科技巨头主导战略合作**：Alphabet（Isomorphic）+ Microsoft（Nuance / Abridge 协同）
- **VC 集中头部**：a16z / GV / Lux Capital / Founders Fund
- **大药企 BD 合作**：Eli Lilly · Novartis · Merck · Pfizer 等与 AI 公司绑定

### 中国
- **港股 + 国资 + 互联网巨头协同**：腾讯 · 字节 · 平安 · 国资基金
- **2023-2024 港股 IPO 窗口**：推想 / 鹰瞳 / 思路迪 等

## 用 [基准率与预期投资](../../99_方法论/框架_基准率.md) 看 Abridge $5B

- $5B 估值 = **未来 3-5 年达到 ≥$500M ARR**
- 美国临床 Copilot 市场：~$3-5B / 年潜在
- 要兑现 $5B：需 **占美国市场 ~10-15%**
- Abridge 已有 100+ 医院系统客户，潜力明显
- **判断**：Abridge 估值不算泡沫——但要看医保 / DRG 支付能否扩大覆盖

## 退出路径预期

| 路径 | 案例 |
|---|---|
| **被大药企并购** | 历史上 AlphaFold 团队被 Isomorphic 整合；Recursion / Exscientia 合并 |
| **被医院系统 / EHR 巨头并购** | Abridge / Nabla 被 Epic / Cerner / Athena 收购可能 |
| **IPO** | Tempus / Recursion / Absci 等已先例；Abridge / Hippocratic 中长期窗口 |
| **持续滚动私募** | Isomorphic Labs 等嵌入巨头型 |
| **失败 / 整合** | 部分单一管线药物发现公司面临关停 |

## 2026 关键变量

1. **Abridge / Nabla 大型医院系统签约扩张** —— Copilot 普及的临床证据
2. **AlphaFold 4 / Isomorphic Labs 临床候选** —— 是否有 AI 设计的药物进入 Ⅲ 期
3. **大药企收购 AI 公司潮** —— Pfizer / Merck / Roche 等是否大动作
4. **Recursion + Exscientia 合并整合** —— 行业整合先例的成败
5. **中国港股第二批医疗 AI IPO** —— 思路迪 / 数坤 等

## 我的判断

> 1. **临床 Copilot 是 2025-2027 估值最高的细分** —— 真实 ROI（医生时间节省）+ 大型医院签约
> 2. **药物发现派会经历持续整合** —— "AI 加速 + 实验室 + 临床"全栈玩家（Recursion + Exscientia / Isomorphic）会胜出
> 3. **2026-2028 中国医疗 AI 港股 IPO 第二波** —— 估值不会回 2018-2021 高峰
> 4. **Tempus 类基因组 + AI 公司面临"基因数据 + AI"双向监管挑战**
>
> **我可能错在哪里**：（a）AI 设计的某药物 Ⅲ 期失败重创估值；（b）大型医院 EHR 自建 Copilot 替代第三方（Epic 自研冲击 Abridge）。
