# Industry Research · 行业研究笔记

聚焦 AI、机器人、具身智能、AI 金融、AI 互联网的行业研究、公司调研与产品调研。

**在线阅读**: https://jeffliulab.github.io/industry-research/

## 姐妹站

- [AI Notes · 人工智能学习笔记](https://jeffliulab.github.io/ai-notes/) — 偏技术原理与课程学习
- Industry Research（本站）— 偏行业·公司·产品的调研笔记

## 本地预览

```bash
pip install -r requirements.txt
python hooks/generate_dropdown.py
mkdocs serve
```

打开 http://127.0.0.1:8000 查看。

## 内容组织

```
docs/
├── 01_AI/          # 大模型、Agent、多模态、AI 基础设施
├── 02_机器人/      # 人形机器人、产业链、核心零部件
├── 03_具身智能/    # VLA、World Model、Sim2Real
├── 04_AI金融/      # 量化、投研、风控、合规
├── 05_AI互联网/    # 搜索、浏览器、AI Coding、应用层
└── 99_方法论/      # 调研框架与信息源
```

每个行业下按「行业研究 / 公司调研 / 产品调研」三个维度组织。
